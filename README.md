<h1 align="center">科亿知识库  KMatrix</h1>

-----------------------------------

项目简介：
-----------------------------------
- 本项目为科亿知识库的重构版本，重新技术选型了Ruoyi-Vue-Plus，第一阶段的目标是先搭建AI平台，借鉴MaxKB、Dify等产品思路。
- 这是前端项目，后端端项目链接：[https://gitee.com/kyxxjs/kmatrix-service](https://gitee.com/kyxxjs/kmatrix-service)

链接与交流互动
-----------------------------------
- 官方站点 ：  [http://www.kykms.cn](http://www.kykms.cn)
- 科亿知识库原版 ：[https://gitee.com/kyxxjs/km_community](https://gitee.com/kyxxjs/km_community)

技术架构：
-----------------------------------
### 后端
- Spring Boot 3.5.7
- Java 17+
- ORM框架: MyBatis Plus 3.5.14
- 支持数据库: MySQL（默认）、Oracle、PostgreSQL、SQL Server
- 缓存: Redis
- 认证框架: Sa-Token 1.44.0（JWT 集成）
- 社交登录: JustAuth 1.16.7

### 前端
- Vue 3.5.25
- Vite 7.2.6
- TypeScript 5.9.3
- SoybeanAdmin
- Naive UI 2.43.2

### 开发环境
- IDEA
- Maven 3.8+
- Node.js: >= 20.19.0
- pnpm： 10+
